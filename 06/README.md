# C/C++ 語言版組譯器的用法

## build

```
csienqu-teacher:06 csienqu$ make clean
rm -f *.o *.exe asm vm dasm
csienqu-teacher:06 csienqu$ make
g++ -std=c++11 asm.cpp -o asm
gcc -std=c99 vm.c -o vm
gcc -std=c99 dasm.c -o dasm
```
